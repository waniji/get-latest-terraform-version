# get-terraform-latest-version

A GitHub Action to get the latest Terraform version.

## Usage

```
- uses: waniji/get-terraform-latest-version@v1
  with:
    # Include prerelease version.
    # (e.g. 1.11.0-alpha20241218)
    #
    # Default: false
    include_prerelease: false
```
